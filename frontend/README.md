# Sweet Shop Frontend

## Setup

1. Navigate to the frontend directory:
```bash
cd frontend
```

2. Install dependencies:
```bash
npm install
```

3. **Important**: Make sure the backend server is running on `http://localhost:3000`

4. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3001`

## Quick Start

After installation, simply run:
```bash
npm run dev
```

## Features

- User registration and login
- Dashboard displaying all sweets
- Search and filter functionality
- Purchase sweets (decreases quantity)
- Admin panel for adding, editing, and deleting sweets
- Admin restock functionality

## Troubleshooting

- **Backend not running**: Make sure backend is running on port 3000
- **Port conflicts**: Vite will automatically use the next available port
- **Module errors**: Run `npm install` again

See `START_GUIDE.md` for detailed instructions.
